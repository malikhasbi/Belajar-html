Software

Android

Android adalah sistem operasi berbasis Linux yang dirancang untuk perangkat bergerak layar sentuh seperti telepon pintar dan komputer tablet. Android awalnya dikembangkan oleh Android, Inc., dengan dukungan finansial dari Google, yang kemudian membelinya pada tahun 2005. wikipedia

Ios

Ios adalah sistem operasi seluler yang dibuat dan dikembangkan oleh Apple Inc. khusus untuk perangkat kerasnya. Ini adalah sistem operasi yang saat ini memberdayakan banyak perangkat seluler perusahaan, termasuk iPhone, dan iPod Touch; itu juga mendukung iPad sebelum pengenalan iPadOS pada 2019. wikipedia

Microsoft Windows

Microsoft Windows atau yang lebih dikenal dengan sebutan Windows saja adalah keluarga sistem operasi. yang dikembangkan oleh Microsoft, dengan menggunakan antarmuka pengguna grafis. Sistem operasi Windows telah berevolusi dari MS-DOS, sebuah sistem operasi yang berbasis modus teks dan command-line. wikipedia

Linuk

Linux adalah keluarga sistem operasi bebas dan sumber terbuka yang pada dasarnya dibangun di atas kernel Linux, yaitu sebuah kernel sistem operasi yang pertama kali dikembangkan oleh Linus Torvalds pada 1991. Linux dirilis di bawah Lisensi Publik Umum GNU versi 2. wikipedia

Github

Gthub adalah layanan hos web bersama untuk proyek pengembangan perangkat lunak yang menggunakan sistem kendali versi Git dan layanan hosting internet. Hal ini banyak digunakan untuk kode komputer. wikipedia

Git

Git adalah perangkat lunak pengendali versi atau proyek manajemen kode perangkat lunak yang diciptakan oleh Linus Torvalds, yang pada awalnya ditujukan untuk pengembangan kernel Linux. Desain Git terinspirasi oleh BitKeeper dan Monotone. wikipedia

Corsair Components

Corsair Components, Inc. adalah perusahaan pembuat perlengkapan dan perangkat keras komputer asal Amerika Serikat yang berkantor pusat di Fremont, California. Perusahaan ini berbadan hukum di California sejak Januari 1994 dan di Delaware sejak 2007. wikipedia
